Project Crobat

A Centalised Searchable Opensource Project Sonar DNS Database

The entire Rapid7 Sonar DNS database indexed, available at your fingertips.

Blazing fast, all Project Sonar DNS data imported into Project Crobat is indexed and updated weekly.

Opensource, all code relating to this project is public on Github

Free, the entire platform, including the Project Sonar API is currently free for everybody to use!

Project Crobat is sponsored by ZeroGuard who kindly provide the hardware, allowing us to keep this service 100% free.
